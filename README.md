Netwronix
Netwronix is an interactive web-based educational portal designed specifically for South African Grade 11 and 12 Physical Sciences students. The platform bridges the gap between theoretical classroom learning and practical application by providing a centralized hub for digital learning.

🚀 Key Features
Live Virtual Classrooms: Integrated video conferencing (via Jitsi Meet) allowing teachers to host real-time lessons filtered by grade.

Dynamic Learning Modules: Organized curriculum content covering Physics topics such as Newton’s Laws, Momentum, and Electrostatics.

Interactive Simulations & Quizzes: Tools designed to test student knowledge and provide hands-on virtual laboratory experiences.

Role-Based Access Control: Distinct dashboards for Teachers (content management and session hosting) and Students (resource access and live session joining).

🛠️ Tech Stack
Backend: Django (Python 3.11)

Database: MySQL (Relational data management)

Frontend: HTML5, Tailwind CSS (Responsive UI/UX)

Communication: Jitsi Meet API integration

🎯 Project Goal
The primary objective of Netwronix is to modernize the science learning experience in South Africa, ensuring that high-level educational resources and live teacher interactions are accessible, secure (POPIA compliant), and organized according to the national curriculum.
